# Statement of Work for Security Assessment with Cygnus Security

**Agreement ID:** SOW-CYGNS-2024-001
**Parties:**
- **Client:** Innovatech Solutions Inc.
- **Auditor:** Cygnus Security Group.
**Effective Date:** January 5, 2024
**Project Term:** January 15, 2024 to April 15, 2024.

## 1. Project Overview
Auditor will perform a comprehensive penetration test and security review of Innovatech's production environment hosted at **Stratus Infrastructure** and the public-facing APIs for **CoreLLM** and **SynapseChat**.

## 2. Scope of Work
- **External and Internal Penetration Testing.**
- **Cloud Configuration Review** against CIS Benchmarks.
- **API Security Testing** for authentication and authorization flaws.

## 3. Fees
A fixed project fee of **Ninety-Five Thousand US Dollars ($95,000.00)**, payable 50% upon commencement and 50% upon delivery of the final report.
