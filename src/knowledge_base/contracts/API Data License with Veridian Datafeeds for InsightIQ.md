# API Data Licensing Agreement with Veridian Datafeeds

**Agreement ID:** DLA-VRDN-2024-005
**Parties:**
- **Licensee:** Innovatech Solutions Inc.
- **Licensor:** Veridian Datafeeds Corp.
**Effective Date:** February 10, 2024
**Term:** Three (3) years.

## 1. Scope of License
Licensor grants Licensee a license to access its real-time and historical financial market data feeds via API for the sole purpose of integration into the **InsightIQ** platform.

## 2. Fees and Usage Tiers
**2.1. Platform Fee:** A flat monthly platform access fee of **Ten Thousand US Dollars ($10,000.00)**.
**2.2. Usage Fee:** A usage-based fee calculated per million API calls per month, with tiers for volume discounts.
**2.3. Data Attribution:** Licensee agrees to display the "Data provided by Veridian Datafeeds" attribution mark within the InsightIQ platform.
