# Hybrid Master Service Agreement with Fusion Robotics

**Agreement ID:** MSA-FUSNR-2024-003
**Parties:**
- **Provider:** Innovatech Solutions Inc.
- **Client:** Fusion Robotics Inc.
**Effective Date:** April 12, 2024
**Initial Term:** Two (2) years.

## 1. Overview
This agreement combines a seat-based license for **NexusFlow** with a usage-based license for the **CoreLLM** API.

## 2. NexusFlow License
**2.1. Scope:** A license for **250 Authorized Users**.
**2.2. Annual Fee:** An annual subscription fee of **$210,000.00**.

## 3. CoreLLM API License
**3.1. Scope:** A license to use the API for R&D data analysis and log summarization.
**3.2. Pricing:** Billed monthly based on tokens processed, with a minimum monthly commitment of **$5,000.00**.
